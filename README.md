## Local notes
This site is built via GitHub Actions, so no local Ruby setup is required.
To update content, edit the markdown files in `_pages/` and navigation in `_data/navigation.yml`.
Images and PDFs should be placed under `assets/` (e.g., `assets/img/`, `assets/pdf/`).
